#####################################
#	     Movement		    #
#####################################
w: move camera forward in the direction it is facing
s: move camera backward in the direction it is facing

#####################################
#	     Zoom		    #
#####################################
a: Zoom out
d: Zoom in

#####################################
#	     Toggles		    #
#####################################
l: Toggles between night and day
k: Toggles wireframe rendering
j: Toggles shading mode
h: Toggles the Heads Up Display


#####################################
#	     Menus		    #
#####################################
Right Click activates the menu

ESC: quits the program
